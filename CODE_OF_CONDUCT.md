# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful, welcoming, and
focused on constructive collaboration.

## Expected Behavior

- Be kind and professional.
- Assume good intent while still being precise.
- Give actionable feedback.
- Respect maintainers' time and project boundaries.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic, hostile, or disruptive comments.

## Enforcement

Maintainers may edit, hide, or remove comments, issues, pull requests, or other
contributions that violate this code of conduct.
